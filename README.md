# Sign-Up Form

A responsive sign-up form built with HTML and CSS as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Features

- Full-page layout with background image sidebar
- Form inputs with HTML validation
- Styled focus states and invalid input feedback
- Custom font integration

## Skills Demonstrated

- Advanced CSS layout and positioning
- HTML form structure and validation attributes
- CSS pseudo-classes (`:focus`, `:user-invalid`)
- Working with background images and overlays
- Git workflow and project organization
- Josh Comeau’s custom CSS reset
